# Nexus-River-View

A comprehensive property management, accounting, and CRM application tailored for the Nexus River View project.

## Features
- **Customer Management:** Track customers, installments, dues, and payments.
- **Accounting:** Full double-entry accounting with ledgers, vouchers, and trial balances.
- **AI Chatbot:** Built-in AI assistant with full database read access to answer queries about the business.
- **Reporting:** Generate PDF and Excel reports for transactions, daily cash, and more.
- **Exporting:** Integrated with Pandas and OpenPyXL to handle structured data exports.
